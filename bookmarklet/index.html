Kader-Import Einrichtung
⚽ Bookmarklet einrichten Empfohlen
1
Lesezeichenleiste einblenden
Falls nicht sichtbar: Strg+Shift+B (Windows) oder Cmd+Shift+B (Mac)
2
Diesen Button in die Lesezeichenleiste ziehen:
⚽ TM → ScoutBoard
Desktop: Button mit der Maus direkt in die Lesezeichenleiste ziehen.
iPad/iPhone: Seite als Lesezeichen speichern → Lesezeichen bearbeiten → URL durch den Code unten ersetzen.
3
Fertig! Das Bookmarklet ist eingerichtet.
📋 CODE FÜR IPAD/IPHONE
✓ Kopiert!
🏃 So importierst du einen Kader
1️⃣ Geh auf transfermarkt.de und öffne die Kaderseite eines Vereins
z.B. transfermarkt.de/first-vienna-fc/kader/verein/2364
2️⃣ Klicke das Bookmarklet ⚽ TM → ScoutBoard in deiner Lesezeichenleiste
3️⃣ Ein Popup erscheint — bestätige oder ändere den Kadernamen
4️⃣ ScoutBoard öffnet sich automatisch — wähle „Ersetzen" oder „Hinzufügen"
✅ Kader ist importiert und gespeichert — fertig!
📋 Alternative: CSV Import
Falls das Bookmarklet nicht funktioniert, kannst du Kader auch per CSV importieren.

Format: Semikolon-getrennt, eine Zeile pro Spieler:
Nummer;Name;Position Beispiel:
1;Unger;TW
4;Szerencsi;IV
7;Monschein;ST In ScoutBoard: 📋 CSV Button im Header klicken → Datei auswählen.
